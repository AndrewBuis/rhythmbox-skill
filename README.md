# rhythmbox-skill
RhythmboxSkill for Mycroft
## About 
Let Mycroft help you control Rhythmbox. When asked Mycroft interfaces with Rhythmbox to play selections from your music library. 

## Examples 
* "play **"
* "play ** playlist"
* "play something by **"
* "shuffle ** playlist"
* "pause"
* "resume"
* "next song"
* "previous song"
* "stop Rhythmbox"
* "play some music by Huey Lewis and the News"
